# TravelExperience(almost done)
TravelExperience is an interactive web application that allows users to track and visualize the countries they've visited around the world. Users can register, login, mark countries as visited, and see statistics about their global travels.


## 🌍 Live Demo

-  [https://worldexpo.netlify.app](https://worldexpo.netlify.app)

## 🛠️ Technologies

- **Java 17**
- **Spring Boot 3.2.5(Security, MVC, Data)**
- **Maven 3.9.6**
- **JavaScript(ThreeJS)**

## 💾 Database
- **MongoDB Atlas** - cloud storage

## Deployment
- **Netlify**: Frontend hosting
- **Railway**: Backend hosting
